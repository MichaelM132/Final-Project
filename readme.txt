In order to run this code download FinalProject.m

Click Run

The button in the botton left clears the graph

The text box on the sides of the axis are just text boxes for the user to enter the desired measurement

The text box above the graph is a place to put a title on the graph

The text box in the bottom of the page directly to the left of the button in the middle is for the x values

The text box in the bottom of the page directly to the right of the button in the middle is for the y values

Enter both x and y values as a comma seperated list EX: 1,2,3,4

The button in the bottom middle basically runs the data and plots it on the graph

the selections on the right allow the user to change the color and shape of the points. In order to switch select desired option and then click button in the bottom middle.
